# Changelog

## 0.1.0

Initial release.

- **Watch mode** (on by default): take a screenshot from any app with the native macOS shortcut (`Cmd+Shift+3/4`) and its path is auto-injected into your Claude Code terminal, even while VS Code is in the background.
- **In-app capture shortcuts** for when VS Code is focused: `Cmd+Option+3` (full screen), `Cmd+Option+4` (region), `Cmd+Option+5` (window).
- **`Cmd+Option+0`** toggles watch mode; status-bar button opens a capture menu.
- Terminal targeting by name, configurable screenshot/save folders, auto-pruning of temp captures.
- No network calls, no telemetry, no dependencies.
